03 Projects

Personal Work

AML Detection on Synthetic Transaction Network Thesis · 2026

Fraud detection on a synthetic AML transaction dataset. Accounts and transactions live in a Neo4j graph; I pulled features with Cypher and trained four classifiers, each with a different imbalance strategy. The corrected pipeline lands at AUC 0.906 on the held-out set.

The headline isn't the AUC. My first version scored F1 = 1.0, which was a red flag, not a result. Chasing it down meant diagnosing three different forms of label leakage, the subtlest being proximity features quietly anchored on test-set landmarks. Rebuilding the feature pipeline and the train/test split to fix it became the actual contribution of the thesis.

Python Neo4j Cypher scikit-learn XGBoost

Weather Data Pipeline & Analytics Personal · 2025

Hourly weather pulls from the OpenWeather API across a few cities. Raw JSON gets parsed into a normalised SQLite schema; there's a notebook on top for KPIs and exploration.

The incremental loading is the bit I'm happy with. Every run backfills the last 6 hours; once a day there's a longer 72-hour pass to catch any retroactive corrections from the API. Upserts on hourly keys mean reruns don't break anything.

Retail Sales Pipeline (PySpark) Personal · 2025

Monthly retail sales CSVs in, partitioned Parquet out. Along the way: address parsing, timestamp normalisation, Window-function dedup. There's an EDA notebook on the cleaned data.

I built this to learn, not to ship. The point was feeling where distributed processing actually differs from pandas, and what partitioning and Parquet are like in practice. Better to figure that out on a toy dataset.
