03 Projects

Personal Work

AML Detection on Synthetic Transaction Network Thesis · 2026

End-to-end fraud detection pipeline on a synthetic financial transaction dataset in an anti-money laundering context. Modelled accounts and transactions as a graph in Neo4j, wrote Cypher queries to extract graph-derived features, and fed them into machine learning classifiers including XGBoost to detect suspicious transaction patterns.

The hardest part: identifying and fixing label leakage caused by graph-neighbour propagation — a subtle issue where transaction features were inadvertently encoding information from neighbouring nodes that wouldn't be available at inference time. Resolving it required rethinking both the feature engineering logic and the train-test split strategy from scratch.

Python Neo4j Cypher scikit-learn XGBoost

Weather Data Pipeline & Analytics Personal · 2025

End-to-end pipeline that collects, stores, and analyses hourly weather data from the OpenWeather API across multiple cities. Raw JSON responses are cleaned, structured, and loaded into a normalised SQLite schema with KPI generation and exploratory analysis on top.

Implemented robust incremental loading: a short backfill covering the last 6 hours runs on every execution, a longer 72-hour backfill runs once daily to capture retroactive API corrections. Upserts on hourly keys make reruns safe and idempotent.

Retail Sales Pipeline (PySpark) Personal · 2025

PySpark pipeline to ingest, clean, deduplicate, and transform monthly retail sales CSV files into partitioned Parquet datasets. Includes address parsing, timestamp normalisation, and Window-function-based deduplication, plus an EDA notebook on the cleaned dataset.

The goal was to understand distributed data processing firsthand: where it differs from single-machine pandas workflows, how columnar storage and partitioning work in practice, and what the trade-offs are — not to solve a real business problem, but to build the muscle memory before it matters.
